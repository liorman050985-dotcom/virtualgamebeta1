# Multiplayer Sprite Game

פרויקט דוגמה למשחק מולטי-פלייר פשוט עם Socket.IO, כולל צ'אט, מוכן לפריסה ב-Render.

## הרצה מקומית
- התקנת חבילות: `npm install`
- הרצה: `npm start`
- גלשו ל- http://localhost:3000

## פריסה ל-Render
- דחפו את הקוד ל-GitHub.
- ב-Render צרו Web Service מהמאגר, או השתמשו ב-`render.yaml` (Infra as Code).
- Render תשתמש ב-`npm install` ו-`npm start`. השרת מאזין ל-`PORT` אוטומטית.
